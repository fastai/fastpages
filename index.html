About Me

This site is built with fastpages, An easy to use blogging platform with extra features for Jupyter Notebooks.

fastpages contain special features for Jupyter Notebooks, such as:

  • Interactive visualizations made with Altair remain interactive.
  • Ability to hide cells (input, output or both).
  • Ability to have collapsable code cells that are either open or closed by default.
  • Embed Twitter cards and YouTube videos.
  • Define the Title, Summary and other metadata via a special markdown cells.
  • Support for local and remote images, including image captions with markdown.
  • Ability to add links to Colab and GitHub automatically.
  • And more.
  • Learn more about how to use these features with this quick tutorial.

fastpages also supports writing blog posts with Microsoft Word documents and Markdown. Additionally, fastpages uses GitHub Actions to completely automate the conversion process. See the fastpages repo for more details.

You can edit the index.md file to change this content.
